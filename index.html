    • Htmlix

    Htmlix - javascript frontend framework based on the data properties of an html document.

    Features:

    • an object-oriented approach to building an application, the presence of a structure and hierarchy of components, the possibility of inheriting component properties,
    • the presence of user events for updating the interface and transferring data between components,
    • the presence of user events for updating the interface and transferring data between components, embedded in a page already given by the server, does not require server rendering,
    • the ability to build applications using a router and html templates,
    • typing properties reduces the number of methods used and reduces the amount of code required for writing and memorizing.

    Htmlix application Examples:

    • Collage_n online editor for creating collages and sprites from images
    • Prototype of the SPA online store
    • tests - test applications for the framework
