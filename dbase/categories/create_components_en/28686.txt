  "text": "Properties are objects that have access to the properties of an html page, and properties can also be event listeners. To create a property, you must specify its\r\nname after the container name, and also specify the type of this property. Let's create some properties: my-class, paragraf and btn-click in the page container:\r\n\r\n\r\n\r\n```html\r\n<style type=\"text/css\">\r\n \r\n .new_class { color: red; }\r\n \r\n</style>\r\n\r\n<div data-page=\"container\"> \r\n\r\n <p data-page-paragraf=\"text\" data-page-my_class=\"class\"> <!-- property of paragraf with type \"text\" and my_class with type \"class\" -->\r\n text\r\n</p>\r\n \r\n <button data-page-btn_click=\"click\">Button</button> <!-- btn_click - property - listener of the \"click\" event -->\r\n\r\n</div>\r\n```\r\n\r\nThe name of the property in html comes after the container name and the \"-\" sign (page-), the \"-\" sign cannot be used in the name of the property itself.\r\nThen after the = sign comes the type of its own, we have three different types: \"text\", \"class\" and \"click\".\r\nIf the property type is an event, then in the application description, in the 'methods` object for given property\r\nyou must specify a method of the same name with an event handler.\r\n\r\n\r\n\r\n\r\nFurther in the description of the application:\r\n\r\n``` javascript\r\nvar StateMap = {\r\n\r\n page: {\r\n container: \"page\",\r\n \r\n props: [\"paragraf\", \"my_class\", \"btn_click\"], //created three properties in the page\r\nmethods container: { \r\n \r\n btn_click: function(){ // method of the same name for the event property;\r\n \r\n console.log(this); \r\n \r\n this.parent.props.paragraph.setProp(\"New text\");\r\n//this.parent - access from a specific property to the container \r\n \r\n this.parent.props.my_class.setProp(\"new_class\");\r\n \r\n              }\r\n          } \r\n       }\r\n  }\r\n```\r\n\r\nSo we have created three properties, one of which is the \"click\" event handler.\r\nthis - in the method indicates the property to which this handler is attached - btn_click.\r\nNext, using .parent.props, we get access to the container and then to all the properties of the container, then by the name of the property to a specific property.\r\nThe `setProp` method is universal and works in different ways."
