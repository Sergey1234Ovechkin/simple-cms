  "text": "Container properties are located in the 'props` object;\r\n\r\nDepending on the type, different fields may be present here.\r\n\r\nFields for all types of properties:\r\n\r\n* `type` - property type in this case is \"click\";\r\n* `htmlLink` - html link to this property;\r\n\r\nIf the property is an event, group, or variant(render-variant):\r\n\r\n* `propName` - property name;\r\n* `parent` - access to the container from this property;\r\n* `pathToComponent` - component name;\r\n* `prop` - any arbitrary data, null by default;\r\n* `rootLink` - root link to the HtmlixState application instance;\r\n\r\nAdditional fields for the `group` type:\r\n* `groupChild` - an array with containers of this group;\r\n* `groupArray` - a reference to a virtual array of containers of this group;\r\n\r\nAn additional field for the `render-variant` type:\r\n* `renderChild` is a reference to the displayed component in this property, or to the container of the virtual array. \r\n\r\nAdditional fields for custom events:\r\n\r\n* `emiter` access to a custom event from a property;\r\n* `emiterKey` is the unique key of the user event listener;\r\n\r\n\r\nAdditional field for standard events:\r\n* `events` - an object with event handler methods for the property, accessed by the event name;\r\n\r\nAdditional field for a property with the data type:\r\n* `parent` - access to the container from this property;\r\n* `propName` - property name;"
