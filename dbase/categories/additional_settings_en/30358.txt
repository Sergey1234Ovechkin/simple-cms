  "title": "Template \"template\" for an array",
  "text": "When creating an array, there must be a container template inside it.\r\nIt is placed in the `.TemplateData` field of the array.\r\nThis template is taken from the first container (with index 0) and stored in this field.\r\nBut what if we want to create an empty array, and only add containers there in the future.\r\nTo do this, an array is created with a template container, which is automatically deleted from it after the array is created.\r\n\r\nFor example:\r\n\r\n\r\n```html\r\n \r\n <div data-some_arr=\"array\">\r\n \r\n <div data-some_cont=\"template\" style=\"display: none;\">\r\n \r\n the template container will be deleted immediately after the application initialization \r\n style=\"display: none;\" in the template itself will change to style=\"\"\r\n\r\n</div>\r\n\r\n</div>\r\n \r\n```\r\n\r\nStyle=\"display: none;\" is specified - so that the template is not visible until it has been deleted."
